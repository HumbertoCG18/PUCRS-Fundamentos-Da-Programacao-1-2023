TÍTULO DO PROJETO: Aplicativo de Gerenciamento de pacientes em um hospital. 

OBJETIVO DO PROJETO: Facilitar o manejamento dos pacientes em um geral, além de ter um maior controle nos medicamentes prescritos para cada paciente. 

VERSÃO ou DATA: Versão 2.0. data 22/06/2023 

COMO INICIAR O PROJETO: 

No VSCode (Windows & Mac): 1 - Executar o GitHub Desktop 
                           2 - Ir em arquivos - Clonar Repositório - URL 
                           3 - Colcoar esta url e clicar em "Clonar" https://github.com/HumbertoCG18/PUCRS-Fundamentos-Da-Programacao-1/tree/main/Exercicios/TF/AppMedicamento 
                           4 - Escolhe a pasta desejada
                           5 - Abra o arquivo, e execute "AplicacaoHospital.java"
                            
No VSCode (Linux): 1 - Abra o terminal no Linux.
                   2 - Navegue até o diretório onde você deseja clonar o repositório. Você pode usar o comando cd para navegar entre os diretórios. 
                   Por exemplo, se você deseja clonar o repositório na sua pasta inicial, digite o seguinte comando: " cd ~ "
                   3 - Agora, execute o comando git clone seguido do URL do repositório. 
                   No seu caso, o comando seria: " git clone https://github.com/HumbertoCG18/PUCRS-Fundamentos-Da-Programacao-1.git " Certifique-se de copiar o URL correto do repositório. 
                   4 - Pressione Enter para executar o comando. O Git irá baixar todos os arquivos e histórico do repositório para o diretório atual.

                   AVISO: Certifique-se de ter o Git instalado no seu sistema antes de executar os comandos acima.

No BlueJ (Windows): 1 - Abra a pasta do arquivo
                    2 - Execute com o botão direito "void main (String []args)"
